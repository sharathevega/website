# TechConf 2026 - Conference Website

A professional conference website for TechConf 2026, featuring information about speakers, schedule, registration, and contact details.

## Features

- **Home Page**: Conference overview with key details (date, location, attendee count, pricing)
- **Schedule**: Complete 3-day conference agenda with keynotes, workshops, and networking events
- **Speakers**: Featured speaker profiles (8 industry leaders)
- **Contact & Registration**: Contact information, registration form with various ticket tiers, and FAQ section
- **Responsive Design**: Mobile-friendly layout that works on all devices
- **Professional Styling**: Modern gradient design with smooth animations and hover effects

## Conference Details

- **Event**: TechConf 2026
- **Dates**: June 15-17, 2026
- **Location**: San Francisco Convention Center, San Francisco, CA
- **Expected Attendees**: 2000+ from 50+ countries
- **Ticket Prices**:
  - Early Bird: $299
  - Standard: $399
  - Premium (All-Access): $599
  - Group (5+): $349 per person

## How to Use

Simply open any HTML file in your browser:

- Double-click `index.html` to open the home page
- Navigate using the menu links throughout the website

## Pages Overview

### Home (index.html)

- Conference tagline and overview
- Key conference details (date, location, attendee info, pricing)
- About section describing the conference
- Highlights of what attendees can expect

### Schedule (schedule.html)

- Three-day detailed schedule
- Each session shows:
  - Time slot
  - Session title
  - Speaker name
  - Location/Room

### Speakers (speakers.html)

- 8 featured speaker cards with:
  - Speaker avatar (emoji placeholder)
  - Full name and title
  - Brief biography
  - Topics of expertise
  - Hover animations for interactivity

### Contact & Registration (contact.html)

- Conference location and contact information
- Registration form with fields:
  - Name, email, phone, company
  - Ticket type selection
  - Interest areas (checkboxes)
  - Additional comments
  - Newsletter subscription
- FAQ section with common questions
- Business hours and social media

## Customization

### Change Conference Details

Edit the following in the HTML files:

- Conference name: Search for "TechConf 2026" and replace
- Dates: Change "June 15-17, 2026"
- Location: Update "San Francisco Convention Center"
- Contact info: Update email and phone numbers

### Modify Speaker Information

Edit the speaker cards in `speakers.html`:

- Change avatar emojis
- Update names and titles
- Modify biographies
- Update topic areas

### Update Styling

Edit `styles.css` to customize:

- Color scheme (modify `:root` CSS variables)
- Typography
- Layout and spacing
- Responsive breakpoints

### Add More Pages

Create new `.html` files and:

1. Add navigation links in the navbar
2. Follow the same HTML structure
3. Include the `styles.css` link
